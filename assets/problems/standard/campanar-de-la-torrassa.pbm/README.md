# The bell tower of La Torrassa

## Features

- Standard problem: read input and write output
- Statements in multiple languages
- Graphics in the statement
- Awards in .`png` and `.html`
- `.ops` file to increase time limit
- Simple simulation too slow (`slow.cc` solution included)
- `loosy` checker that converts PEs to ACs

## Abstract

Count the number of bell rings during a certain lapse of time.
