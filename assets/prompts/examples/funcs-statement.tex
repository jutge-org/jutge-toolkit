\Problem{Primes everywhere}

\usePython

\Statement

Carla is a young girl who loves math. She is interested in prime numbers and wants some functions to help her in her mathematical musitations.

\medskip

\begin{enumerate}

    \item Write a function r\lstinline|is_prime(n: int) -> bool| that returns True if $n$ is a prime number, and False otherwise.

    \item Write a function r\lstinline|next_prime(n: int) -> int| that returns the smallest prime number greater than $n$.

    \item Write a function r\lstinline|count_primes(a: int, b: int) -> int| that returns the number of prime numbers between $a$ and $b$ inclusive.

\end{enumerate}

All given values by Carla are non-negative integers.
